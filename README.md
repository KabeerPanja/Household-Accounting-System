# 📊 Personal Accounts Management System

A web-based personal accounting application built using Python and Streamlit.
This app helps users manage monthly balances, track expenses, and view financial summaries in a simple and user-friendly interface.

- 🚀 Features

- 🔐 User login system

- 📅 Month-wise balance management

- 💸 Add, edit, and delete expenses

- 📈 Monthly financial overview

- 🔍 Search and filter expenses by month and date

- 💾 Data stored using JSON

- 🎨 Clean and easy-to-use UI

# 🛠️ Technologies Used

- Python

- Streamlit

- JSON (for data storage)

# ▶️ How to Run the Project Locally

- Install Streamlit:

- pip install streamlit

## Run the application:

- streamlit run app.py

- Open the displayed URL in your browser.

# ⚠️ Data Usage Note

This repository uses dummy data for demonstration.

If you want to use this application for personal accounting:

Replace data/expense.json with your own private data.

Do not upload your real financial data to public repositories.
